# 📊 Log Analyzer (Bash)

Simple Bash script to analyze server logs and extract basic metrics.

## 🚀 Install ans Usage

```bash
git clone https://github.com/4ntoniomj/Nginx_Log_Analyser
cd Nginx_Log_Analyser
chmod +x script.sh
./script.sh [log_file]
```
